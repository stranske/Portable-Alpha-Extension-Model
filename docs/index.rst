Portable Alpha Extension Model Documentation
===========================================

Welcome to the Portable Alpha Extension Model documentation!

This project implements a Monte Carlo framework for portfolio allocation across three "sleeves":
Internal, External Portable-Alpha, and Active Extension strategies.

Quick Start
-----------

.. code-block:: bash

   # Setup development environment
   ./dev.sh setup

   # Run demo
   ./dev.sh demo

   # Start dashboard
   ./dev.sh dashboard

Key Features
------------

* Monte Carlo simulation framework
* Three-bucket capital allocation model
* Comprehensive risk metrics
* Interactive Streamlit dashboard
* Extensive visualization suite
